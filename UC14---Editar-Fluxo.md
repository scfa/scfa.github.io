#### 1. Caso de Uso: UC14 - Editar Fluxo

## Descrição

Este caso de uso permite a edição de um Fluxo.

## Ator

Usuário

##Pré-Condições

Usuário deverá está logado no sistema. Pelo menos um Fluxo deverá estar criado.

## Fluxo Básico 
1. O Usuário escolhe a opção de Visualizar Fluxo.
2. O usuário escolhe a opção de Editar Fluxo.
3. O Sistema exibe o Fluxo atual.
4. O Usuário seleciona a disciplina que gostaria de alterar a sua posição no fluxo.
5. O usuário seleciona no checkBox o número do semestre a qual gostaria de alocar a disciplina.
7. Os passos 4 e 5 são repetidos quantas vezes necessárias pelo usuário.
8. O usuário clica no botão de editar Fluxo.
9. O caso de uso é encerrado.
	
## Fluxos Alternativos
* Não há.

## Pós Condições
* Plano de Fluxo editado.

## Fluxo de Excessão
* Não há.
